## 🔥 **Project Overview**

### Introduction
- **in**spiration + **space**
- 다양한 영감으로 채우는 나만의 공간

### Background
- 시각적, 청각적 매체를 한 번에 검색하여 스크랩하기 어려움
- 자유로운 배치를 통해 나만의 스크랩북을 제작하고자 함

### Goals
- 여러 곳에 흩어져 있는 매체를 한 곳에 모아 공감각적인 공간 제작 가능
- 맞춤형 스크랩북 형식으로 사용자만의 독창적인 공간 제작 가능
